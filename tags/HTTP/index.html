HTTP 协议的特点

  1. 支持客户/服务器模式

  2. 简单快速：客户向服务器请求服务时，只需要传送请求方法和路径，请求方法常用的有 GET，POST，HEAD 等，每种方法规定了客户与服务器联系的类型不同，由于 HTTP 协议简单，使得 HTTP 服务器的程序规模小，因而通讯速度很快
