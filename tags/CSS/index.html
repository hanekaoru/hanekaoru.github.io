BFC

BFC 直译为 “块级格式化上下文”，它是一个独立的渲染区域，只有 Block-level box 参与， 它规定了内部的 Block-level Box 如何布局，并且与这个区域外部毫不相干

块级元素会自动生成一个块级盒 block-level box，这是块级盒 block-level-box 的盒模型构成，它表明的是块级盒自身的结构构成
