函数的 length 和 callee 属性

需要注意的是，ES6 以后不再提倡使用函数的 callee 属性

函数的 length 是形参列表的长度，就是函数定义的时候写在 () 中的参数个数，无视实参的个数，也就是说，arguments.callee.length 就是形参列表的个数，而 arguments.length 则表示实参个数
