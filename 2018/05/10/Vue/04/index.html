axios

先来看看一些 axios 的特点

  • 浏览器端发起 XMLHttpRequest 请求

  • node 发起 http 请求

  • 支持 Promise API

  • 拦截请求和返回

  • 转化请求和返回（data）

  • 取消请求

  • 自动转化 json 数据

  • 客户端支持抵御 XSRF

Request Config

请求时的设置只有 url 是必须的，如果没有指明 method 的话，默认的请求方式是 GET

Response Schema
