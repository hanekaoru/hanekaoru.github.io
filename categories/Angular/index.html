这里主要涉及到三个 API

  • ViewChild 一个属性装饰器，用来从模版视图中获取对应的元素，可以通过模版变量获取，获取的时候可以通过 read 属性设置查询的条件，简单来说就是可以把视图转为不同的实例

  • ComponentFactoryResolver 一个服务，动态加载组件的核心，这个服务可以将一个组件实例呈现到另一个组件视图上
