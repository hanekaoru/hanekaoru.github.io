JavaScript 中的 || 和 &&

因为在 JavaScript 中，它们返回的 并不是 布尔值，它们的返回值是两个操作数中的一个（且仅一个），即选择两个操作数中的一个，然后返回它的值（其定义可以参考 二元逻辑运算符）
