.map()

基本定义

map 方法会给原数组中的每个元素都按顺序调用一次 callback 函数

  • callback 每次执行后的返回值组合起来形成一个新数组

  • callback 函数只会在有值的索引上被调用，那些从来没被赋过值或者使用 delete 删除的索引则不会被调用

  • callback 函数会被自动传入三个参数，数组元素，元素索引，原数组本身
